= OpenShiftのアーキテクチャ
:navtitle: OpenShiftのアーキテクチャ

include::vars.adoc[]

image::common-environment-ocp-architecture.png[OpenShift Architecture]

このワークショップはパブリッククラウド上で稼働している OpenShift 環境でホストされています。環境は以下のシステムで構成されています：

* コントロールプレーンノード
* ワーカー（または「アプリケーション」）ノード
* 動的プロビジョニングストレージ
* プライベートコンテナレジストリ
* ルーティング層
