== 概要

このワークショップでは、開発者の視点からOpenShiftを実践的に学びます。実践的なインタラクティブ演習を通じて、OpenShiftプラットフォーム上でコンテナ化されたアプリケーションを構築、デプロイ、管理する方法を習得します。

== 学習内容

*OpenShiftの基礎:*

* OpenShiftのコマンドラインクライアント（CLI）とWebコンソールの使用
* OpenShiftのDeveloper（開発者）インターフェースの操作と主要な概念の理解

*アプリケーションのデプロイと管理:*

* 既存のコンテナイメージを使用したアプリケーションのデプロイ
* Gitリポジトリのソースコードからの直接アプリケーションをデプロイ
* アプリケーションラベルを使用したコンポーネントの識別と整理
* 様々なトラフィック負荷に対応するためのアプリケーションのスケーリング

*ネットワーキングとアクセス:*

* ルート（Route）を介した外部ユーザーへのアプリケーションの公開
* アプリケーションネットワーキングとサービスディスカバリの設定

*モニタリングとトラブルシューティング:*

* デバッグのためのアプリケーションログの表示と分析
* 直接的な操作とトラブルシューティングのためのアプリケーションコンテナへのアクセス

*データベース連携:*

* OpenShiftのサービスカタログからのデータベースのデプロイ
* データベースに接続して使用するためのアプリケーションの設定

*コラボレーションと自動化:*

* 共同開発のための他ユーザーへのアクセス権付与
* コード変更からの自動アプリケーションビルドを可能にするWebhookの設定

NOTE: このワークショップではJavaアプリケーションを使用しますが、ここで学ぶ概念とテクニックは、.NET、Node.js、Python、その他のランタイムで構築されたアプリケーションにも適用できます。
